# Augmented Reality Light Engine

This project involves the complete design and development of an Augmented Reality (AR) Light Engine module. It showcases expertise in high-density electronics, bridging hardware architecture with advanced PCB manufacturing techniques. 

Core Architecture of the project implemented in KiCad: 

-**Powered by a DPLC2020 ASIC (ARM Cortex M3) and a DLPA2020 PMIC** to manage complex power rails.

-**Optical Engine**: Integrates a DLP2020 MEMS projector with OSRAM RGB LEDs to generate high-fidelity AR visuals.

-**Rigid-Flex Technology**: Designed on a 6-layer Rigid-Flex PCB to fit the ergonomic constraints of wearable smart glasses.

-**Signal & Power Integrity**: Implementation of 50Ω single-ended and 100Ω differential pairs for high-speed video data, alongside optimized switching loops for EMC compliance.
