# Circuit for controlling the concentration of CO 

This project crates an automated safety system designed to maintain indoor CO levels between 400–6000 ppm.  

-**Pipeline**: Integrates a resistive gas sensor, linear voltage converter, and a hysteresis comparator for stable switching.  

-**Actuation**: Uses nMOS current amplification to drive an orange LED and a ventilation relay.  

-**Validation**: Optimized for mass production via Monte Carlo and Worst Case stability analyses.
