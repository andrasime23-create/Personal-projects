# Digital Image Processing

A Python-based implementation for real-time sign recognition featuring:

-**Day/Night Adaptability**: Automated frame classification using mean intensity statistics for optimized segmentation.

-**Invariant Matching**: Log-polar coordinate transformation and Normalized Cross-Correlation (NCC) for scale and rotation-independent detection.

-**Advanced Dimensionality Reduction**: Feature extraction using Neighborhood Preserving Embedding (NPE).

-**Robust Classification**: High-precision recognition powered by a Support Vector Machine (SVM).
